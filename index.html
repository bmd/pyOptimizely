pyOptimizely

A minimal python wrapper for the Optimizely REST API.

Note: This project has become more of a demo for correctly building a Python application that utilizes a Travis/Coveralls CI stack. It's way overengineered for what it is an you should use something like Tortilla for wrapping a simple REST API like Optimizely's.

Example

from optimizely import Optimizely

OPTIMIZELY_TOKEN = 'abc123'

opt = Optimizely(OPTIMIZELY_TOKEN)
experiments = opt.get('/experiments').json()

print(json.dumps(experiments, indent=2))
